How labels work

  • Leave Page Range empty to label all pages.
  • Use styles like Roman numerals for front matter and decimal numbering for the main body.
  • Use Prefix with Start Value 0 to create labels such as A-0, A-1, A-2.
  • Enable progress when one rule uses a disjoint range such as 1-9,30-40 or odd.

Frequently Asked Questions

What is a page label?

A page label is the numbering shown by PDF readers in thumbnails, page navigation, and print dialogs. It does not print new text onto the page itself.

Can I mix Roman numerals and normal numbers?

Yes. Add multiple rules, such as 1-4 with LowercaseRoman and 5-20 with DecimalArabic.

Do my files stay private?

Yes. The processing happens in your browser using CoherentPDF WASM. Your PDF does not leave your device.
